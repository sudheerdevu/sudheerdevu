### About Me

I'm a **GPU Performance & AI Systems Engineer at AMD**, based in Hyderabad. I work across the full AI compute stack — from writing Linux kernel modules and HIP GPU kernels to optimizing LLM inference on AMD Instinct and Ryzen AI hardware.

- Working on **ROCm, MIGraphX, ONNX Runtime** optimization for AMD GPUs
- Building tools for **AI inference benchmarking & performance profiling**
- Writing **HIP kernels** (attention, GEMM, reduction) and **Linux device drivers**
- Expertise in **model-to-hardware optimization** across CPU, GPU, and NPU backends
